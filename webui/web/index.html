Подключение в игре

Игра хранит список подключённых модов отдельно (ModCFG). Здесь можно синхронизировать его со сборкой.

Журнал

SR Mods Launcher репозиторий токен

Настройки

Туда, где лежит Rangers.exe. Моды ставятся в подпапку Mods.
Проверить, не вышла ли новая версия лаунчера (сравнивает с релизом в репозитории).
Дополнительно (база, репозиторий и токен) — обычно менять не нужно
Лагерь определяется автоматически по файлам в папке Mods и показан над списком. Здесь его можно задать вручную — только как пометку для переносимой сборки, когда игра ещё не установлена. На установку не влияет.
Нужен только для приватного репозитория. Оставьте пустым, если не знаете, что это.
Накладываются поверх основного по приоритету (верхний — выше). Форк может заменить или добавить отдельные файлы мода; моды, которых нет в основном репозитории, не ставятся.

Подтверждение

Сборки модов

Нажмите на сборку, чтобы переключиться (текущая отмечена). Окно остаётся открытым.
Добавить в текущую сборку все моды одной метки (целый лагерь). Потом — «Установить все».
Сборка — это переносимый набор: база + список модов. Можно отдать другому игроку.

Добавить в сборку

Любой лагерь сразу. У каждого мода — метка лагеря.
Какую ветку модов берём. Можно добавить весь лагерь целиком.
«Весь лагерь» — добавить все паки. Иначе выберите конкретный пак.
«Весь пак» — добавить пак целиком. Иначе — один мод из пака.
Прямая ссылка на файл <id>.json форка мода (GitHub/HF).
