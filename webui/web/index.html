Настройки

Туда, где лежит Rangers.exe. Моды ставятся в подпапку Mods.
Проверить, не вышла ли новая версия лаунчера (сравнивает с релизом в репозитории).
Тихий режим (по умолчанию): бесконфликтные обновления применяются автоматически, без окна — только итог-уведомление. Окно всплывает, лишь когда нужно ваше решение. Включите галочку, если хотите видеть план перед каждым обновлением.
Дополнительно (база, репозиторий и токен) — обычно менять не нужно
То же, что селектор «Сборка» в панели над списком. Определяет базовую сборку, относительно которой идёт проверка обновлений. «Авто» — определить по файлам в Mods (кнопка 🎯 в панели).
Если установлены моды из нескольких сборок — добавьте их сюда. Обновление каждого мода проверяется по первой сборке списка, где мод есть (один раз). Строка 1 — базовая сборка.
Нужен только для приватного репозитория. Оставьте пустым, если не знаете, что это.
Накладываются поверх основного по приоритету (верхний — выше). Форк может заменить или добавить отдельные файлы мода; моды, которых нет в основном репозитории, не ставятся.

Нажмите на профиль, чтобы переключиться (текущий отмечен). Окно остаётся открытым.
Добавить в текущий профиль все моды из одной сборки. Потом — «Установить все».
Профиль — это переносимый набор: база + список модов. Можно отдать другому игроку.

Любая сборка сразу. У каждого мода — метка сборки.
Какую ветку модов берём. Можно добавить всю сборку целиком.

Справка

Коротко о том, как пользоваться лаунчером. Полная инструкция с картинками — по кнопке «Полная справка на GitHub» ниже.
1. Папка игры. Укажите папку, где лежит Rangers.exe (клик по адресу игры в шапке или в Настройках). Моды ставятся в подпапку Mods.
2. Профиль. Это сохранённый набор модов. Держите несколько и переключайтесь селектором «Профиль» в шапке; кнопка ＋ — создать/удалить.
3. Добавить мод. Кнопка «➕ Добавить мод»: искать по названию, выбрать по цепочке Сборка → Пак → Мод, или добавить по ссылке. Можно добавить целую сборку.
4. Установить. «🧩 Установить все» скачивает и ставит моды профиля. Галочка «В профиле» у строки включает/выключает мод.
5. Обновления. «⟳ Проверить обновления» сверяет ваши моды с сервером, «🔀 Обновить все» скачивает новые версии, сохраняя ваши ручные правки.
6. База. Селектор «База» — с каким сборником сверяться при обновлениях. Оставьте «авто», если не уверены; 🎯 — определить сейчас по файлам в Mods.
7. Играть. Кнопка «▶ Играть» запускает Space Rangers HD. Панель справа синхронизирует список подключённых в игре модов (ModCFG) с профилем.
