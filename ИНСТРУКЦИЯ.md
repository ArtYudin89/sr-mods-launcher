# SR Mods Launcher — инструкция для игрока

Это пошаговое руководство для того, кто видит лаунчер впервые. Лаунчер помогает выбирать
наборы модов для **Space Rangers HD: A War Apart**, скачивать их и ставить в игру.

---

## 0. Что нужно заранее

- Установленная игра **Space Rangers HD** (с файлом `Rangers.exe`).
- Файл лаунчера **`SRModsLauncher.exe`** (скачивается из раздела Releases репозитория).
- **GitHub-токен** — нужен, только если репозиторий с модами приватный (по умолчанию он
  приватный). Как получить: GitHub → Settings → Developer settings → Personal access tokens →
  создать токен с правом `repo` (чтение). Если пользуешься публичным форком репозитория —
  токен не нужен.

Просто запусти `SRModsLauncher.exe` — установка не требуется. Рядом с ним появятся файлы
настроек (`launcher_config.json`) и профилей (папка `profiles/`).

---

## 1. Первый запуск — указать игру

Вверху, в блоке **«Настройки»**:

1. Поле **«Игра:»** → нажми **«Обзор»** и укажи папку, где лежит `Rangers.exe`
   (или сам `Rangers.exe`). Лаунчер сам найдёт подпапку `Mods`.
2. Если репозиторий приватный — вставь свой **GitHub token** в поле справа
   («GitHub token (для приватных/агрегатора)»).

Кнопка **«📂 Папка модов»** открывает папку `Mods` игры в проводнике (удобно проверить).

---

## 2. Профиль = твой набор модов

**Профиль** — это сохранённый набор модов. Можно держать несколько (например «Чистый Редакс»,
«Со всеми модами», «Только Юнивёрс»).

- **Новый** — создать профиль. **Сохранить** — сохранить текущий. **Удалить** — убрать.
- Переключение профиля меняет список модов ниже. Профиль `default` удалить нельзя.

---

## 3. Добавить моды в список

Слева — список модов. Кнопки над ним: **＋** добавить, **－** убрать, **↑/↓** порядок,
**⟳** проверить обновления.

Нажми **＋** — откроется окно. Вверху выбери **Тип**:

- **Мод (дескриптор)** — *рекомендуется*. Нажми **«⟳ Каталог»**, дождись загрузки и выбери мод
  из списка «Мод из каталога». Зависимости подтянутся сами при установке.
  - «Источник (опц.)» — если один и тот же мод есть в разных паках, можно выбрать конкретный.
  - «URL форка (опц.)» — если хочешь поставить чей-то форк мода: вставь прямую ссылку на его
    файл-дескриптор (`<id>.json`).
- **Юнит агрегатора** — поставить целый пак или один мод из пака. Заполни repo/camp/unit,
  кнопка **«⟳ Список»** покажет моды внутри пака. Поле «Мод (пусто=весь)»: пусто = весь пак.
- **Generic ZIP** — просто ссылка на zip с модом (распакуется в `Mods`).

Нажми **«Добавить»**. Повтори для всех нужных модов.

В списке у каждого мода есть **Статус**: `⏳` — ещё не скачан, `✅` — установлен,
`⬆ обн.` — вышло обновление.

---

## 4. Собери корректный набор (важные правила)

Моды в Space Rangers ставятся слоями. Запомни три правила:

1. **Ровно один «базовый пак».** Это полная сборка игры (содержит `Rangers.exe`): например
   **Редакс**, **Оридж (Original)** или **Юнивёрс**. Нельзя смешивать две базы в одном наборе.
2. **Фикс-пак ставится только на свою базу.** Например фиксы Редакса нужны вместе с базой
   Редакса. Базовые и фикс-паки — критичные: если для них вышло обновление, его нужно ставить.
3. **Сейвы привязаны к базе.** Если начал партию на одной базе (скажем, Коммунити), посреди
   игры сменить базу (на Редакс) нельзя — старые сейвы не подойдут. Новую партию — пожалуйста.

---

## 5. Проверь совместимость — кнопка «🛡 Проверить совместимость»

Перед установкой нажми **«🛡 Проверить совместимость»** (блок «Действия» справа). Лаунчер
проверит набор и покажет отчёт:

- `⛔` — в наборе **несколько баз** (так нельзя — оставь одну).
- `⚠` — в наборе **нет базы**, или **фикс без своего родительского пака**.
- `💾` — предупреждение: выбранная база **отличается от установленной** → сейвы текущей партии
  не подойдут.
- `❗` — список базовых/фикс-паков (их обновления обязательны).
- `✅` — всё в порядке.

Это только подсказка — лаунчер ничего не блокирует, решение за тобой.

---

## 6. Установка

В блоке **«Действия»** справа:

- **🧩 Установить набор (с зависим.)** — *основной способ*: ставит все моды из списка и
  автоматически подтягивает зависимости.
- **⬇ Установить выбранный** — поставить только выделенный в списке мод.
- **⬇ Установить все** — поставить все по очереди.
- **🗑 Очистить Mods** — очистить папку `Mods` игры (например, перед сменой набора).

Внизу справа — **полоса прогресса** и **лог** (что происходит). Код и графика модов
качаются автоматически (графика — с публичного хранилища, без токена).

---

## 7. Запустить игру

Нажми **«▶ Запустить игру»** вверху. Лаунчер запустит `Rangers.exe` с установленным набором.

---

## 8. Обновления

Нажми **⟳** над списком модов — лаунчер сверит версии с репозиторием. У кого вышло обновление,
получит статус **`⬆ обн.`**. Переустанови такой мод (или весь набор), чтобы обновиться.
Обновления базовых и фикс-паков ставить **обязательно**.

---

## Частые вопросы

- **Ничего не грузится / пусто в каталоге.** Проверь GitHub-токен (для приватного репозитория)
  и интернет.
- **Сменил базу — пропали сейвы.** Это ожидаемо (правило 3). Старую партию продолжай на старой
  базе; для новой базы начни новую игру.
- **Где лежат настройки.** Рядом с `SRModsLauncher.exe`: `launcher_config.json` и `profiles/`.
- **Как сменить оформление.** Положи свой `theme.json` рядом с `.exe` (см. README).
